Their own company reports

Any free public Innovatr reports that are deliberately exposed to everyone (if that is supported)

Implementation guidance:

On the API side:

Check all app.get("/api/member/reports"...), /api/member/client-reports, /api/member/studies, or similar member endpoints in server/routes.ts.

For each member facing endpoint, use the authenticated session user (req.session.user or equivalent) and filter by:

companyId (preferred) or

user email that maps to a company

Do not allow an email query parameter to override the company when the user is not admin. If any member endpoint accepts ?email= or ?companyId=, ignore those for non admin and always derive from the session.

Admin endpoints like /api/admin/reports and /api/admin/client-reports must remain protected with requireAdmin and can still return all data.

On the frontend side:

For the client My Research page, make sure it calls the member endpoints that are scoped to the current session user, not the admin endpoints.

For the admin My Research or Admin → Reports, use the admin endpoints.
